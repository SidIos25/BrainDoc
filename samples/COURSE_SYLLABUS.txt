COURSE DESCRIPTION

This advanced graduate seminar explores state-of-the-art techniques in deep learning and modern machine learning architectures. Topics include convolutional neural networks (CNNs), recurrent neural networks (RNNs), transformer models, attention mechanisms, and reinforcement learning. Students will implement models from recent research papers and apply them to real-world datasets.

PREREQUISITES

- CS 401: Machine Learning Fundamentals (or equivalent)
- Linear Algebra
- Python programming proficiency
- Basic knowledge of calculus and probability

COURSE OBJECTIVES AND LEARNING OUTCOMES

Upon successful completion of this course, students will be able to:

1. Understand the mathematical foundations of deep learning and neural network optimization
2. Design and implement various neural network architectures (CNNs, RNNs, Transformers)
3. Apply deep learning techniques to solve classification, regression, and sequence prediction problems
4. Understand and apply attention mechanisms and transfer learning
5. Implement reinforcement learning algorithms
6. Evaluate and critique deep learning models using appropriate metrics
7. Understand current research trends and limitations of deep learning
8. Implement state-of-the-art models from academic papers

COURSE CONTENT AND TOPICS

Week 1-2: Review and Fundamentals
  - Neural networks and backpropagation review
  - Optimization methods (SGD, Adam, RMSprop)
  - Batch normalization and regularization techniques

Week 3-5: Convolutional Neural Networks (CNNs)
  - CNN architecture and convolution operations
  - Classic architectures (LeNet, AlexNet, VGG, ResNet)
  - Image classification applications

Week 6-8: Recurrent Neural Networks (RNNs)
  - RNN, LSTM, and GRU architectures
  - Sequence modeling and time series prediction
  - Vanishing gradient problem

Week 9-10: Attention Mechanisms and Transformers
  - Attention mechanisms and self-attention
  - Transformer architecture
  - BERT and GPT models
  - Applications in NLP

Week 11-12: Transfer Learning and Domain Adaptation
  - Pre-trained models and fine-tuning strategies
  - Domain adaptation techniques
  - Few-shot learning

Week 13-14: Reinforcement Learning
  - Markov Decision Processes
  - Q-Learning and Deep Q-Networks (DQN)
  - Policy Gradient Methods

Week 15: Advanced Topics and Emerging Research
  - Generative Adversarial Networks (GANs)
  - Variational Autoencoders (VAEs)
  - Neural Architecture Search (NAS)
